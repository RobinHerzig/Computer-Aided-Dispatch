First Line Response

Several years ago, I accepted an amazing opportunity to work in Mcmurdo Research Station, Antarctica with the Antarctica Fire Department.

Frustrated by the lack of an adequate CAD (computer-aided dispatch) program, I sought to create my own. First with Microsoft Word, using content controls to add functionality. However, as the project grew, I explored JavaScript and other languages to create something awesome.

After several revisions, I'm happy to release my CAD software so that it may help agencies who cannot afford their own. Furthermore, I'm excited to continue my journey as a software engineer by taking on new projects and solving new problems.
